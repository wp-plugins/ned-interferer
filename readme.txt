Plugin to place an interferer onto your Wordpress Blog

== Description ==

NED interferer lets you place an interferer, meaning an image linked to an url just somewhere within your blog. The interferer will be overlayed to anything below.

== Installation ==

1. Upload `ned_stoerer` folder to your WP plugin folder `/wp-content/plugins/` directory
2. The path must look like this: `/wp-content/plugins/ned_stoerer/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. To make changes, go to Settings, NED interferer.

Thanks for installing!
