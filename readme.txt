Plugin to place an interferer onto your Wordpress Blog. This plugin does not need any theme modifications as it overlays the interferer onto your side

== Description ==

NED_stoerer lets you place an interferer, meaning an image linked to an url just somewhere within your blog. The interferer will be overlayed to anything below.

This plugin does not need any theme modifications as it overlays the interferer onto your side. The interferer image can be uploaded directly to the wordpress media library within the plugin admin interface. Optionally any already uploaded image can be selected from the media library.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

Thanks for installing.
